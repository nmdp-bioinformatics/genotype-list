This is a portal which will allow a user to authenticate using OpenID.

It is based on the Spring Security OpenID sample application version 3.2.x
https://github.com/SpringSource/spring-security/tree/master/samples/openid

The UI uses Spring MVC and was generated with Spring Roo.

It has been modified to build with Maven and act as the OAuth 2.0 authorization provider for the GL Service.
