<br/>
== Usage ==

<p>Accession sort.</p>
{{{
$ gl-accession-sort --help
usage:
accession-sort [-i identifiers.txt] [-o sorted-glstrings.txt]

arguments:
   -h, --help  display help message [optional]

   -i, --identifiers [class java.io.File]  identifier input file [optional]

   -o, --glstrings [class java.io.File]  sorted glstring output file [optional]

   -d, --output-identifiers  include identifiers in output [optional]

   -g, --output-glstrings  include original glstrings in output [optional]

   -s, --output-sorted-glstrings  include sorted glstrings in output [optional]
}}}
